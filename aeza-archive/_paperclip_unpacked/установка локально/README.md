# Paperclip AI — локальная установка

Панель управления AI-агентами. Запускается локально на вашем Mac,Linux или Windows

---

## Требования

- Node.js **v22+** (через nvm: `nvm install 22 && nvm use 22`)
- pnpm **v9.15+**: `npm install -g pnpm`
- Подписка Claude **Pro или выше** (нужна для Claude Code CLI)

---

## Способ 1 — npx (не рекомендую)

```bash
npx paperclipai onboard --yes
```

Запустится автоматически: скачает пакет, создаст БД, откроет дашборд.

**Почему не рекомендую:** все файлы — конфиги, база данных, логи — раскидываются по системным кэшам npx. Найти их потом сложно, почистить — ещё сложнее. Если что-то пойдёт не так, вы просто не поймёте куда смотреть.

---

## Способ 2 — клонировать репозиторий (рекомендую)

Всё лежит в одной папке, вы контролируете каждый файл.

**1. Клонируй репозиторий:**
```bash
git clone https://github.com/paperclipai/paperclip.git
cd paperclip
```

**2. Установи зависимости:**
```bash
pnpm install
```

**3. Запусти:**
```bash
pnpm dev
```

Дашборд откроется на `http://localhost:3100`.

При первом запуске автоматически создаётся встроенная PostgreSQL — ничего настраивать не нужно.

---

## После запуска: авторизация Claude Code

Без этого агенты не работают.

**1. Открой новый терминал и запусти:**
```bash
claude
```

**2. Войди в аккаунт:**
```
/login
```

Откроется ссылка — логинишься в браузере через Anthropic аккаунт. Нужна подписка **Pro или выше**.

**3. Перезапусти Paperclip** (Ctrl+C → `pnpm dev`), чтобы он подхватил авторизацию.

---

## Где лежат файлы

| Путь | Что |
|------|-----|
| `~/.paperclip/instances/default/config.json` | Основной конфиг |
| `~/.paperclip/instances/default/db/` | PostgreSQL данные |
| `~/.paperclip/instances/default/data/storage/` | Файлы агентов |
| `~/.paperclip/instances/default/logs/` | Логи сервера |
| `~/paperclip/` | Исходники (если клонировал) |

---

## Полезные команды

```bash
# Запуск с живой перезагрузкой
pnpm dev

# Запуск без наблюдения за файлами
pnpm dev:once

# Только API-сервер (без фронтенда)
pnpm dev:server

# Сборка
pnpm build
```

---

## Удаление

```bash
# Данные и конфиги
rm -rf ~/.paperclip

# Исходники (если клонировал)
rm -rf ~/paperclip
```
