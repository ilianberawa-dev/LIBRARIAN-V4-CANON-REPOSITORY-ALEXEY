# Материалы Алексея Колесова

**Источник:** Приватный Telegram канал Алексея  
**Экспорт:** 2026-04-20  
**Всего:** 420+ файлов с Aeza + локальные материалы

## Структура

- `transcripts/` - Транскрипты аудио/видео (8+ файлов)
- `media/` - PDF, фото, документы из канала
- `metadata/` - JSON индексы (library_index.json, p4_catalog.json)
- `guides/` - PDF гайды

## Ключевые файлы

- `metadata/library_index.json` - Индекс 142+ постов с метаданными
- `INDEX.md` - Общий индекс материалов
- `Алексей Колесов - Private.md` - Sync файл канала

## Использование

Материалы организованы по типам. Для поиска конкретной темы используй library_index.json.
