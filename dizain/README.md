## Как добавить новый дизайн-проект

### Через алиас (быстро)
```bash
upload-design <путь_к_файлу>   # копирует в dizain/inbox/
```

### Вручную
1. Положи файл/папку в `dizain/inbox/`
2. В чате роли 4 скажи: "разбери inbox"
3. Чат переместит в `projects/<имя_проекта>/` с `README.md`
