# 🏗️ АРХИТЕКТУРА — настройка AI-чатов и промптов

**Тема:** как устроены роли, промпты, протоколы общения, многоагентные схемы.

---

## Что сюда попадает

- Промпты для Claude / GPT / других моделей
- Схемы ролей и иерархий (Архитектор / Прораб / Работяга)
- Идеи "как настроить AI чтобы он делал X"
- Скриншоты разговоров с AI (если поучительные)
- Описания multi-agent систем

## Структура

```
arhitektura/
├── README.md
└── projects/
    └── <имя-проекта>/    ← все файлы одного проекта вместе
```

## Как сюда попадают файлы

Только через `inbox/` и команду «разбери inbox». Чат классифицирует как «архитектура» → перекладывает сюда в `projects/<имя>/`.
