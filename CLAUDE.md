## 📋 ПРАВИЛА ПОВЕДЕНИЯ

- **Имя пользователя — Илья.** Алексей Колесов — автор канонической базы
  (коуч Ильи), не пользователь чата. Не путать.
- **Канон vs чат.** Канон = только `kanon/` + `alexey-materials/` (артефакты
  автора с фронт-матерами `author:` / `source_msg_id:`). Решения и фразы Ильи
  в чате — это запросы **пользователя**, **НЕ канон**. В коммитах и
  session-memory различать: «решено в чате с Ильёй» vs
  «согласно `kanon/<file>:<line>`».
- **Только факты + альтернативы.** Не pitch, не маркетинг-стиль.
- **Краткость в чате.** Ответ длиннее ~3 КБ — спроси разрешения разделить.
  Полные файлы кода в чат не показывай — делай Edit/Write, в чат summary.
- **Читай только нужное:** `kanon/`, `CLAUDE.md`, `.claude/skills/*.md`,
  `alexey-materials/metadata/index_compact.json`. Без команды не делай
  `find`/`ls`/`grep` по подпапкам.
- **Коридор разрешений** в `~/.claude/settings.json`:
  `rm*`, `sudo`, `git push --force`, `git reset --hard` — **запрещены**.
  `git push`, `npm publish`, `docker build` — **спрашивают**.
