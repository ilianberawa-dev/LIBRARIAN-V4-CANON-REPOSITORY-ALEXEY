**Автор экстракта:** `librarian-v2` (продолжение v1, UI: setup-tg-parser)

**Источник:** `library_index.json` — 142 активных поста приватного канала Алексея Колесова

**Назначение:** учебный артефакт для school-v<N> и других ролей — быстрый маппинг «модуль школы L0–L7 → топ-5 постов канона».

---

## Как читать

- Модули L0–L7 соответствуют curriculum школы (см. `school_manifest.json`).
- Для каждого модуля — топ-5 постов/материалов канона, отранжированных по ценности для обучения.
- ✅ = транскрипт / материал уже доступен локально в `docs/alexey-reference/`.
- 🕑 = в очереди скачивания/транскрибации на Aeza (`download_p4_whitelist`).
- Ссылки `msg_<id>` разрешаются через `library_index.json` → заголовок, kind, дата, путь к файлу.
- Это **экстракт, не замена `library_index.json`**. Для полного поиска (142 поста) — всегда `library_index.json`.
