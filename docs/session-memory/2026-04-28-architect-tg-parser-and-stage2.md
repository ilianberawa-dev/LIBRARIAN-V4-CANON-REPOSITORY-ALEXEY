## Ключевые решения

1. **Канон vs чат:** канон = только `kanon/` + `alexey-materials/`. Слова Ильи в чате — НЕ канон.
2. **Видео в репо — никогда.** Только транскрипты + метадата + PDF. Pipeline: download → transcribe → delete.
3. **`telegram_url` обязательна и не пустая.** Если не получили — STOP, не сохранять, логировать.
4. **`_INDEX.md` живёт вместе с библиотекой.** Парсер обновляет строку `alexey-materials/` после каждого транскрипта.
5. **Squash merge + cherry-pick от свежего main** (не rebase с накопленными коммитами).
