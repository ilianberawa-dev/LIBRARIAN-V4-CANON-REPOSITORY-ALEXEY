### 4. Архитектурный выбор по 12 принципам

| Принцип | За file-based | За LightRAG |
|---|---|---|
| #0 Простота | ✅ 1-2 шага | ⚠️ 6-8 шагов |
| #3 Simple Nodes | ✅ файлы независимы | — |
| #5 Fail Loud | ✅ явные ошибки | ⚠️ чёрный ящик |
| #8 Validate Before Automate | ✅ Илья сутки уже на ней | требует Этап 4 (контрольный тест) |
| Memory Pyramid | file-based закрывает Уровни 1-3 | LightRAG нужен для Уровня 4 (большой корпус) |

Из 12 принципов 4 сильно за file-based, 0 за LightRAG, 8 нейтральны. **НО Memory Pyramid + позиция Алексея говорят: для уровня 4 (поиск по 154 постам + 44 транскриптам) нужна семантика, а не grep.**
