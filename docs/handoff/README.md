# Папка `docs/handoff/`

Личная передача памяти от одного чата конкретному преемнику.

**НЕ читать обычным чатам.** SessionStart hook сюда **не смотрит** — handoff передаётся только когда Алексей в первом сообщении нового чата вставляет JSON со ссылкой на конкретный файл.

## Отличие от `docs/session-memory/`

| Папка | Кому | Когда читать |
|---|---|---|
| `docs/session-memory/` | Всем чатам (через hook) | Всегда при старте |
| `docs/handoff/` | **Только наследнику** | Только по явной команде через JSON |

## Формат имени файла

`<YYYY-MM-DD>-<role>-<topic>.md` — то же что в session-memory, но смысл другой: это полная передача контекста для одного чата.
