## A/B test protocol — FROZEN until v2 re-run

**When we re-test (after applying canon-compliance changes):**
1. **Same 21 records** (или extended set если удастся добрать tanah/apartemen/ruko)
2. **Same 4 models** через LiteLLM
3. **Same prompt skeleton** for CLS-002/003 — но с канонической SKILL.md секцией и bilingual triggers
4. **Compare metrics table above** — accept if Grade A ≥ current AND tokens ≤ +10% AND agreement ≥ current

**Reject change if:**
- Grade A снизится (consolidation broke the signal)
- Agreement упадёт ниже 80%
- Token cost взорвётся >20% (bilingual descriptions usually smaller, but check)
