# 📥 INBOX — единая свалка для новых материалов

**Что это:** одна папка. Кидай сюда всё новое — макеты, тексты, ссылки, картинки.

**Зачем:** не надо помнить 5 папок. Помнишь одну.

---

## Как сюда попадают файлы

1. **Перетащи в файл-менеджере** — открой папку `inbox/` в репо, перетащи файл
2. **Через ярлык на рабочем столе** — `inbox (3)` показывает счётчик файлов
3. **Через обычного Клода** — попроси его сделать PR в `inbox/`

---

## Как разобрать

В любом чате Claude Code:
> «разбери inbox»

Чат:
1. Спросит: «Это один проект или несколько?»
2. Определит тему: дизайн / архитектура / прочее+идеи
3. Создаст папку проекта в правильной теме
4. Переложит **ВСЕ файлы одного проекта в ОДНУ папку** (не разбрасывая по типу)
5. Покажет отчёт «положил X в Y, согласен?»

---

## Триггеры что пора разобрать

- **При открытии чата** — он сам скажет «в inbox 3 файла, разобрать?»
- **Ярлык на рабочем столе** — показывает число `inbox (3)`
- **Telegram-уведомление** — бот пингует когда в inbox что-то прилетает (если настроено)

---

## Правила

- ⛔ Не клади сюда секреты (пароли, токены) — публичный репо
- ⛔ Не клади архивы > 50 МБ — лучше ссылку на облако
- ✅ Файл удаляется отсюда после разбора
