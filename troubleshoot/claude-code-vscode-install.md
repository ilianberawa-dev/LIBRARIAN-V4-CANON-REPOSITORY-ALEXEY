# Установка Claude Code Extension в VS Code

Альтернатива Kilo Code, если он тормозит. Официальное расширение от Anthropic.

---

## Когда использовать

- Kilo Code тормозит / не подключается к claude
- Нужна стабильная интеграция Claude в VS Code
- Хочется быстро попробовать Claude без отдельного терминала

---

## Установка — 3 шага

1. **Открой Extensions** (иконка квадратиков в боковой панели VS Code)
2. **Введи в поиск** `Claude Code`
3. **Нажми Install** на расширении от **Anthropic** (`anthropic.claude-code`)

---

## Требования

- VS Code **1.98.0** или выше
- Размер: ~165 МБ
- Identifier: `anthropic.claude-code`

---

## После установки

1. **В правом верхнем углу появится иконка запуска** (значок Claude)
2. **Кнопка "Ask Claude to edit..."** — основное окно ввода задач
3. **Параллельно может работать Kilo Code** в боковой панели

> ⚙️ Совет: если хочешь вернуться в терминальный режим —
> *"Prefer the Terminal experience? Switch back in Settings."*
